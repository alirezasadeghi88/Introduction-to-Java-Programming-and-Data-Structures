# Chapter 1: Introduction to Computers, Programs, and Java

This chapter introduces the fundamental concepts of computer science, programming languages, and the basics of the Java programming language.

## Learning Objectives
- [x] Understand computer basics, programs, and operating systems (§§1.2–1.4).
- [ ] Describe the relationship between Java and the World Wide Web (§1.5).
- [ ] Understand the meaning of Java language specification, API, JDK™, JRE™, and IDE (§1.6).
- [ ] Write a simple Java program (§1.7).
- [ ] Display output on the console (§1.7).
- [ ] Explain the basic syntax of a Java program (§1.7).
- [ ] Create, compile, and run Java programs (§1.8).
- [ ] Use sound Java programming style and document programs properly (§1.9).
- [ ] Explain the differences between syntax errors, runtime errors, and logic errors (§1.10).

## Coding Exercises
All source code for this chapter's exercises is located in this directory:
- `Welcome.java`: A simple program to display a greeting message.
- `WelcomeWithThreeMessages.java`: Displaying multiple lines of text.
- `ComputeExpression.java`: Practicing basic mathematical evaluations.

## Key Concepts
> **Note:** Java is platform-independent. The Java compiler translates source code into **bytecode**, which is then executed by the **Java Virtual Machine (JVM)**.
